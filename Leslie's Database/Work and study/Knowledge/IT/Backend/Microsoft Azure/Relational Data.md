[[Relational Data]]

###Characteristics of Relational Data
-Data is stored in a table (consisting of Rows and Columns)
 Each Row represents a specific entity.
 Each Columns represents the property of the entity.
 Each column represents a specific data type 
 All rows have the same number of columns
[[Primary Key]]: specific identifier of a row.
[[Foreign key]]: reference or Link to the primary key of another table. 
                     (used to create a reference or Link between tables).
[[Normalization]]: Split an entity into more than one Table.


Structure
[[clustered Index]]: Sorting data on a particular column.
[[Non-clustered Index]]: creating a table(Sorting) with the new data in a arranged manner.
[[Views]]: A virtual table based on the result of a query
