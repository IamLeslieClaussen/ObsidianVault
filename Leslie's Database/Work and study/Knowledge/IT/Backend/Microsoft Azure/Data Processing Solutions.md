###[[Data Process]] Solution:



[[OLTP]](Transaction System):
Day-to-Day handling of Transactions that result from an enterprise operation
-Small, discrete, unit of work.
-Often high volume
-Data processed very quickly




[[OLAP]](Analytical Solutio):
Analysis of Data stored in a database for the purpose of making management decisions
-Big picture view of the data(information) held in a database.
-Gives insight to make business decisions.


[[Data Ingestion]]: Data collection from different sources into one central location.
it can be in a form of [[Batch ]] or [[Stream]]
[[Data Process]]: The transformation of the ingested data. Eg Filtering or Simple Transformation. Transformation into a meaningful format. Data cleaning, Data rendering
