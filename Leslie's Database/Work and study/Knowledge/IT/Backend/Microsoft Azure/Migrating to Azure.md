
Tools:

[[azure database migration service]], [[DMS]]:
enables easy migrations from multiple on premise or other cloud services. Offers both online and offline migrations methods. During an offline migration, the Date will be unavailable.
Online migration can help reduce downtime by continues sync between source and destination. 
If downtime is not acceptable use the online migration method. 
[[Data migration]], [[DMA]]:  Helps you access on premis migration and informs you about compatability issues.
[[tranactional replication]] is supported to an Azure SQL Database or Managed Instance.
