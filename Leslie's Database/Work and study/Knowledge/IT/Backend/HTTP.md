Hyper Text Transfer Protocol 

It is broken down into 4 parts;


| HTTP       | REQUEST                                     | RESPONSE       |
| ---------- | ------------------------------------------- | -------------- |
| Start line | Version HTTP 1, Method, eg "search? g=tuna" | ", Status code |
| Headers    | Host: eg www.google.com, TOKEN              | Cookies        |
| Blank line | contains nothing just used as seperator     |                |
| Body       | Username and pw                             | HTML webpage   |
|            |                                             |                |
